# Readium Web Monorepo

DISCLAIMER: This is not the official Readium Web monorepo. This is just a sample.

## Getting Started

### Prerequisites
- Node.js

### Install development dependencies

```
npm install
```

### Build all packages from source

```
npm run build
```

### Run all tests

```
npm run test
```

### Build in watch mode

```
npm run watch
```
